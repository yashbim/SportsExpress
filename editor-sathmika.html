I am Kulunu Sathmika 1st year Software Engineering student. The first task I was assigned, is to create the main site navigation and the Home page. This home page was designed to access several parts of the website such as, About us, Gallery, Quiz and Sitemap and so on. According to the theme of the website, I have created a query form. so if all values are not filled out, the page will generate a alert saying that the relative form field should be filled out. Once all required form input fields are filled in with values, the user can continue by clicking the "View query" button. once it's clicked, the summary of the user inputs will be displayed along with two other buttons consisting with "Submit" and "Edit" buttons. so when user clicked on the "edit" button, it redirects to the form which you have filled in and the user can edit their responses. And, when you click "send" button, it will email your form details to the developer's email address.
